## 1. 디렉토리

- cd : 디렉토리로 이동

cd [상대경로]
cd java   # 현재 디렉토리에 있는 java 디렉토리로 이동

상대경로

# 10. 그 외 선택자
카상 클래스, 가상 요소

## 10.1. 동작에 따른 가상 클래스

- :link  (방문 안한 곳)
- :visited  (방문 한 곳)
- :hover (호벌)
- :active (마우스 위로 올렸을 때 뭐시기..)
- :focus (아 집중집중 ㅠㅠ)

## 10.2. 태그의 상태에 따른 가상 클래스

- : enabled - 활성화 된 상태
- : disabled - 비 활성화
- : checked - 체크된 상태

## 10.3. 구조 가상 클래스
- :root - 문서 전체
- :not - 부정
- : nth-child(n) 
  - n 에 2n-1 , odd
  - n 에 2n, even
